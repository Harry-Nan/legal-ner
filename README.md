# Legal NER

This repository contains resources, code, and data for exploring **Named Entity Recognition** (NER) in the legal domain.

## Repository Structure

- [Annotation protocol [DUTCH].pdf](https://github.com/Harry-Nan/legal-ner/blob/main/Annotation%20protocol%20[DUTCH].pdf)  
  PDF detailing annotation guidelines for legal NER for Dutch Administrative Decisions.

- [prompt.md](https://github.com/Harry-Nan/legal-ner/blob/main/prompt.md)  
  Markdown file describing the used prompt to generate LLM annotations.

- [data/](https://github.com/Harry-Nan/legal-ner/tree/main/data)  
  Directory containing datasets used to train NER(RE) models.

- [results/](https://github.com/Harry-Nan/legal-ner/tree/main/results)  
  Output directory for storing more detailed results of the models.

- [sentence selection/](https://github.com/Harry-Nan/legal-ner/tree/main/sentence%20selection)  
  Output directory for regular expressions related to the sentence selection techniques.
