# NER Dataset

This repossitory contains the **training** and **testing** data used to train NER models.

## Label dictionary

| Token | Value |
|----------|----------|
| 0    | o (empty)   |
| 1-2    | Recipient (Ontvanger)  |
| 3-4    | Authority (Besluitvormend Orgaan)    |
| 5-6    | Legal Action (Rechtshandeling)    |
| 7-8    | Legal Object (Rechtsobject)   |
| 9-10    | Legal Basis (Wettelijke Grondslag)    |
